# Madinaty - Apartment Booking System

## 🚀 Features
- Apartment booking with availability validation
- Dynamic pricing calculation
- Admin dashboard
- Localization (EN/AR)

## 🧠 Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
